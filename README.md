# Simple-Diabetes-Prediction
This project is about diabetes predicition using Naive Bayes, KNN and Decision Tree model. This project is created using Flask. Make sure Flask is install in your machine before running this project. This project also build using python version > 3.0.0. To make sure there is no compatibility issue, please run using python 3 and above.

This project already consits .sav file for the models. To create new model, just run the model.py file
## Step to run
To run this project, simply run 'python main.py' in the terminal.

## To create new model
To create new model file (.sav), simply run 'python model.py' in the terminal.
